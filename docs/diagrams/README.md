# Diagram Source Files

This directory contains Mermaid diagram source files for the Boring Company Tunnel Vision bid:

- **vortex-cascade-topology.mmd**: load-bearing structure map
- **sydney-tunnel-route.mmd**: physical tunnel path
- **wave-atom-sphinx-flow.mmd**: integration sequence
- **traffic-coherence-heatmap.md**: before/after comparison
- **multi-ai-orchestration.mmd**: Grok + Claude + Repos
- **fibonacci-weighting.md**: priority visualization
- **sphinx-gates.mmd**: 5-gate verification flow

Each diagram can be rendered via GitHub's built-in Mermaid support or mermaid.live.
